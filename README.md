# SiteForge 🚀

SiteForge is an AI-powered website builder that generates website layouts and content from user prompts. Users can describe the type of website they want, and the application uses AI to generate a customized website structure.

## Features

* AI-powered website generation
* User authentication with Firebase
* Secure payments with Razorpay
* Responsive and modern UI
* MERN stack architecture
* AI integration through OpenRouter (DeepSeek)

## Tech Stack

### Frontend

* React.js
